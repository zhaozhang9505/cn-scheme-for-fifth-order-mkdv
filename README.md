# Numerical study of anomalous interactions in the fifth-order mKdV equation

This repository contains the MATLAB source code for the numerical experiments in Sections 3, 4, and 5 of the paper.

## File organization

The scripts are organized according to the corresponding sections of the manuscript:

- `Sec_3_1.m`, `Sec_3_2.m`  
  MATLAB codes for the numerical experiments in Section 3

- `Sec_4_1.m`, `Sec_4_2.m`, `Sec_4_3.m`, `Sec_4_4.m`  
  MATLAB codes for the numerical experiments in Section 4

- `Sec_5_1.m`, `Sec_5_2.m`, `Sec_5_3.m`, `Sec_5_4.m`  
  MATLAB codes for the numerical experiments in Section 5

## Notes

- All scripts are written in MATLAB.
- The parameters in each script correspond to those used in the paper.
- These files are intended to reproduce the main numerical results reported in Sections 3, 4, and 5.
